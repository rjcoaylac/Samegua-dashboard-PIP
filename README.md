# Dashboard Presupuestal Municipalidad Distrital de Samegua

Dashboard interactivo con 3 vistas navegables (Ranking, Funciones, Proyectos), inspirado en el diseño proporcionado. Cada vista replica el layout con KPIs superiores, gráficos de barra, pie/donut, gauges y progresión mensual.

## Mejoras:
- Navegación por tabs en header.
- Gauges para avances totales y programados (usando doughnut modificado).
- Responsive: Grids adaptables para móvil/laptop.
- Colores pastel suaves con buen contraste.
- Modo oscuro.
- Export CSV/PDF (agregado en script, pero no visible en tabs para simplicidad).
- Datos adaptados de capturas, con placeholders donde faltan (ej. progresión mensual).

## Cómo desplegar en GitHub Pages
1. Crea repositorio.
2. Sube: index.html, data.js, script.js, README.md
3. Settings → Pages → Source: main → Save.
4. Accede: https://tuusuario.github.io/repo/

Actualiza data.js con nuevos datos MEF.
