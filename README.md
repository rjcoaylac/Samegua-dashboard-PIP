# Dashboard Presupuestal Municipalidad Distrital de Samegua

Dashboard interactivo con datos de ejecución presupuestal y avances de proyectos (MEF - corte marzo 2026).

## Cómo desplegar en GitHub Pages

1. Crea repositorio nuevo en GitHub.
2. Sube los archivos: index.html, data.js, script.js, README.md
3. Ve a Settings → Pages → Source: Deploy from a branch → main → Save.
4. Espera 1-2 min → accede en https://tuusuario.github.io/nombre-repo/

Actualiza data.js cuando tengas nuevos datos del portal MEF.

¡Éxitos con la gestión!
